Fullstack
=======  

Airbnb clone site 
-----------
*[Ben Awad - youtube build along course](https://youtu.be/kuswGIH-Xj8)*
Project uses yarn workspaces to bring together web, mobile and the backend.  


Web:
  * react
  * react router
  * **typescript**
  * apollo  

Mobile:
  * react-native
  * react router
  * **typescript**
  * apollo  

Backend:
   * node
   * express/ graphql
   * graphql
   * redis
